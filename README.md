# 🔍 Metodología de Pentesting que practico

En los laboratorios intento seguir una metodología similar a la utilizada en entornos profesionales:

1. Reconocimiento
2. Enumeración de servicios
3. Identificación de posibles vulnerabilidades
4. Explotación
5. Escalada de privilegios
6. Documentación del proceso

Documentar cada paso me ayuda a **consolidar conocimientos y crear guías útiles para otros estudiantes de ciberseguridad**.
